Research

I have always been interested in subjects that have to do with human beings - how we learn, think and behave. When my younger brother had his first suicide attempt, I struggled to understand why he reacted the way he did, despite our same parent. Upon chance, I read Thinking Fast and Slow by Daniel Kahneman and Nudge by Richard Thaler. I learned about the abundance of cognitive biases that could be combatted with empiricism and system 2 thinking. That is when I decided to switch into data science enhance my technical skills and be grounded in the realistic constraints of economics, with the end-goal of applying these knowledge in psychology.

Overconfidence

Overconfidence in Amazon Rekognition

  • Testing for overconfidence displayed by Amazon Web Services' Rekognition API in generating object label confidence ratings
  • Utilizing AWS Lambda and S3 with the Rekognition API to generate object labels for 1500+ images to test for overconfidence against MTurk reviewers

Overconfidence in Google Vision

  • Testing for overconfidence displayed by Google Vision's Cloud API in generating object label confidence ratings against MTurk human and RA reviewers
  • Employed Google Cloud’s Vision API, a pre-trained machine learning model for computer vision, to illustrate overconfidence displayed by 9000+ object associated confidence ratings against MTurk reviewers

Priors vs. Desires
